# Smart Office Seat Occupancy Demo App

A frontend-only demo web app for a smart office seat occupancy system.

## Features

- Live floor map with color-coded seat status
- Hover details for occupied seats
- Summary counts and occupancy rate
- Floor selector with auto-simulated refresh every 10 seconds
- Employee directory with search and live in-office count
- Admin panel with login, manual seat status control, and mock employee management

## Run

Open `index.html` in your browser.
